jquery.jsの作成

本フォルダにファイルを追加することでマージされます。

ファイル名は「jquery」で始まっている必要があります。その直後の文
字がハイフン（-）の場合は本体、ピリオド（.）の場合はプラグインで
あると識別します。拡張子は「js」です。

本体、プラグインのファイル名の順序でマージされます。

各ファイルの文字セットはUTF8で改行文字はLFとします。また、ファイ
ルの最後の改行は必要ありません。実際にはマージの処理で挿入されま
す。また、改行やスペースが除去されているスクリプトが望ましいです。

マージの処理中に拡張子「tmp」の作業用ファイルが作成されます。
何ならかの問題でこれらのファイルが残ることが考えられます。バージョ
ン管理のリポジトリに含めないようにして手動で削除してください。
